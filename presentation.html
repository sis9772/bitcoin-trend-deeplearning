AI 기반 암호화폐 추세 예측 모델

데이터 기반 매매 보조 전략 개발

프로젝트 배경

  • 암호화폐에 대한 관심도 증가
  • 투자보다는 투기에 가까운 성격
  • AI 기반 예측 기술의 중요성 증대
  • 군중의 매매 심리는 통계를 따름

핵심 기술

  • 딥러닝 모델 (LSTM) 활용
  • 핵심 기술 지표 직접 추출 및 선정
  • 매매 전략에 따른 라벨링 최적화
  • 시계열 데이터 전처리 및 학습

핵심 전략

  • 완벽한 예측보다 추세 판단에 집중
  • 수익률 기준 이진 분류로 보조 지표로 활용
  • 찌라시, 루머에 영향을 덜 받는 단기 매매위주

기대 효과 및 비전

  • 안정적인 매매 보조 지표 역할
  • 감정적 매매를 배제한 의사결정 지원
  • 실제 사용자의 매매 전략에 따라 최적화된 솔루션 제공 가능

데이터 구성 및 특징

효과적인 학습을 위한 피처 엔지니어링

데이터 기본 구성

데이터 출처

Kaggle 비트코인 1분봉 데이터

OHLCV 시계열 구성

(시가, 고가, 저가, 종가, 거래량)

전처리 과정

1분봉 → 15분봉 리샘플링

단기 추세 유지, 노이즈 완화

입력 피쳐

"Open", "High", "Low", "Close", "Volume", "rsi_golden_cross", "macd_cross_up", "bb_contraction", "adx_strong", "bb_break_upper"

BTC기준 신뢰도 높은 피쳐 선택

피처 설계

  • 단순 수치 → 매매 시그널로 변환
  • 인간 트레이더의 해석 방식 모방
  • 경험적인 신뢰도가 높은 지표 중심 구성

기술적 지표 기반 피처 구성 (주요 5종)

RSI 골든 크로스 (매수 시그널)

# 단기 RSI > 장기 RSI
rsi_golden_cross = 1

MACD 상향 돌파 (매수 시그널)

# MACD > 시그널선
macd_cross_up = 1

볼린저 밴드 수축 (변동성 수렴)

# 밴드폭 <= 0.1
bb_contraction = 1

볼린저 상단 돌파 (상승 추세)

# 종가 > 상단 밴드
bb_break_upper = 1

ADX 강세 신호 (추세 뚜렷)

# ADX > 25
adx_strong = 1

피처 유효성 검증

→ 중복되거나 다른 피쳐와 매우 유사한 피쳐들은 T-SNE으로 시각화하여 제거

→ 모델의 구조 단순화 및 학습 시간 타협

→ 초기 96개의 피처를 10개로 간소화

정답(Label) 구성 전략: 시계열 분류 문제 설계

모델 성능의 핵심, '무엇을 예측할 것인가'에 대한 고민

초기 시도: 다중 클래스 분류

수익률을 여러 구간(3~5 Class)으로 세분화하여 라벨링을 시도했습니다.

문제점 발생

  • 라벨 불균형: 횡보(±0.5%) 구간에 데이터가 과도하게 집중되어 모델이 편향적으로 학습했습니다.
  • 모호한 경계: 클래스 간 경계가 애매하여, 해당 구간에서 모델의 예측 성능이 크게 저하되었습니다.
  • 노이즈 학습: 의미 없는 미세한 변동까지 학습하려다 보니 전체적인 예측 성능이 불안정해졌습니다.

최종 전략: 이진 분류로 단순화

문제의 본질에 집중하여, 명확한 추세만 학습하도록 라벨을 재정의했습니다.

라벨 정의 기준

수익률 > +0.5% 상승 (1)
-0.5% ~ +0.5% 노이즈 구간 (학습 제외)
수익률 < -0.5% 하락 (0)

✅ 도입 효과

  • 학습 안정성 대폭 향상
  • 추세가 뚜렷한 시점만 학습하여 정확도 및 실용성 증가

모델 구조 실험 및 최종 선정

복잡성에서 벗어나 본질에 집중한 선택

다양한 모델 구조 실험

초기에는 LSTM, GRU, Transformer 등 다양한 최신 시계열 모델을 적용하여 성능을 극대화하고자 했습니다.

한계점 봉착

  • 자원 부족: 피처 수를 96개까지 확장하고 모델을 깊게 쌓자 RAM 부족 문제가 발생했습니다.
  • 과적합 (Overfitting): 복잡한 모델과 Attention 구조는 훈련 데이터에만 과도하게 최적화되어 안정적인 학습에 실패했습니다.

성능 향상의 핵심 요인

모델 단순화

과적합을 방지하고 학습 속도를 개선하여 안정적인 수렴을 유도했습니다.

핵심 피처 선별

KDE 플롯, t-SNE 등으로 클래스 간 분포 차이가 큰 주요 신호 피처만 채택하여 노이즈를 줄였습니다.

최종 선정 모델: 단일 BiLSTM

실험 결과, 복잡한 구조보다 간결하고 안정적인 단일 BiLSTM 구조가 가장 효과적이었습니다.

Input Layer
60 Timesteps × 10 Features
BiLSTM Layer
128 Units (양방향 처리)
Dropout Layer
Rate: 0.2 (과적합 방지)
Fully Connected
2 Units (Softmax), Cross-entropy

"복잡한 모델이 좋은 성능을 보장하지 않음"

BiLSTM 모델 학습 결과

모델 성능 및 유효성 검증

학습 과정 분석

  • 학습되기 시작: Epoch 5 이후 과적합 억제, 훈련/검증 격차 감소
  • 유의미한 성능: 단순 예측(50%)을 상회하는 최종 검증 정확도 73.15% 달성

검증 세트 성능 (Confusion Matrix)

핵심 결과

  • 전체 정확도 (Accuracy): 75.5%
  • 상승 예측 정밀도 (Precision): 75.3%
  • 하락 예측 정밀도 (Precision): 75.7%
  • 시세를 완벽히 예측하기에는 부족하지만 추세 분석 정도는 유의미하게 할 수 있을 정도의 성능

실용적 가치와 나아갈 방향
